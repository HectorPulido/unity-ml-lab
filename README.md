# 🧬 Unity ML Lab

**Machine learning experiments in Unity — each folder is a self-contained project with its own README.**

![Engine](https://img.shields.io/badge/Engine-Unity-222c37?logo=unity&logoColor=white)
![Language](https://img.shields.io/badge/Language-C%23-178600)
![License](https://img.shields.io/github/license/HectorPulido/unity-ml-lab?color=orange)

Most of these lived as standalone repos; they were merged here to keep the Unity + AI experiments in one place.

| Preview | Experiment | Description |
|:---:|---|---|
| <img src="imitation-learning/Images/ExampleImage.gif" width="180"> | **[imitation-learning](imitation-learning)** | Record the game state and the player's actions, then train a neural network to imitate them |
| | **[genetic-algorithms](genetic-algorithms)** | Three genetic algorithms: function minimization, the Travelling Salesman Problem and more |
| <img src="pong-policy-gradients/Images/gif.gif" width="180"> | **[pong-policy-gradients](pong-policy-gradients)** | Policy gradients (reinforcement learning) applied to Pong, with neural network and softmax approaches |
| | **[dlf-reinforcement-learning](dlf-reinforcement-learning)** | Reinforcement learning in Unity using [DLF/TAS](https://github.com/HectorPulido/Machine-learning-Framework-Csharp), the from-scratch autograd framework |
| <img src="llama2-binding/img/LlamaUnity.gif" width="180"> | **[llama2-binding](llama2-binding)** | Connect Unity with the Llama 2 LLM through Python websockets — client/server, useful for games |
